# Quote Generator

A simple quote generator powered by OpenAI's GPT-4o-mini.

## Setup

1. Deploy to Vercel
2. Add environment variable: `OPENAI_KEY`
3. Visit your deployment URL

## Environment Variables

```bash
vercel env add OPENAI_KEY
# Paste your OpenAI API key when prompted
```

## Development

```bash
vercel dev
```

Then visit http://localhost:3000/apps/quote-generator/
